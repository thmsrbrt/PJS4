Routes :

GET /utilisateurs

Attribut Type Description Exemple
id int Identifiant unique d'un utilisateur http://localhost:3000/utilisateurs?id=1

GET /utilisateurs/all

Retourne tous les utilisateurs de la table utilisateurs.

POST /utilisateurs

Attribut Type Description
nom string Requis : Nom de l'utilisateur
prenom string Requis : Prénom de l'utilisateur
email string Requis : email sous format *@*.*
type string Requis : 'Candidat' ou 'Entreprise'
description string Description du profil de l'utilisateur.
photo string Lien photo de profil.
cvfile string Lien du fichier CV.

DELETE /utilisateurs/:id

Attribut Type Description Exemple
:id int Requis : ID de l'utilisateur à supprimer http://localhost:3000/utilisateurs/2
